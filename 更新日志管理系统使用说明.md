# 更新日志管理系统使用说明

## 功能介绍

为了方便内部人员快速发布产品更新日志，我们开发了一个简单的管理后台。通过这个系统，你可以：

- ✅ 无需修改代码，直接在网页上发布更新日志
- ✅ 粘贴文本内容，自动解析并发布
- ✅ 实时更新，发布后立即在更新日志页面显示

## 使用方法

### 第一步：访问管理后台

打开浏览器，访问：`https://你的域名/admin/changelog`

### 第二步：输入密码

输入管理员密码（请向技术团队获取）

### 第三步：粘贴更新内容

按照以下格式粘贴更新内容：

```
V1.8.0 2026.5.6
新增
支持生成 HTML 格式报告
来源详情支持飞书多张图片展示
支持通过 Agent 聊天生成报告
优化
修复了一些 Bug，提升用户体验
```

**格式说明**：
- 第一行：版本号 + 空格 + 日期（如：`V1.8.0 2026.5.6`）
- 第二行：`新增` 或 `✦ 新增`
- 接下来几行：新增功能列表（每行一个功能）
- 然后一行：`优化` 或 `⚙ 优化`
- 接下来几行：优化内容列表（每行一个优化）

**注意**：
- 如果没有新增功能，可以不写"新增"部分
- 如果没有优化内容，可以不写"优化"部分
- 但至少要有一个部分

### 第四步：点击发布

点击"发布"按钮，系统会自动：
1. 解析你粘贴的内容
2. 保存到数据库
3. 跳转到更新日志页面
4. 新版本会显示在最顶部

### 第五步：查看效果

发布成功后会自动跳转到更新日志页面：`https://你的域名/docs/support/changelog`

你可以看到刚刚发布的新版本已经显示在页面顶部。

## 常见问题

### Q1：密码是什么？
A：请联系技术团队获取管理员密码。

### Q2：格式写错了怎么办？
A：如果格式不正确，系统会提示"格式错误"。请检查：
- 第一行是否包含版本号和日期
- 是否有"新增"或"优化"标题
- 每个功能/优化是否单独一行

### Q3：可以修改已发布的内容吗？
A：目前系统只支持添加新版本，不支持修改已发布的内容。如需修改，请联系技术团队。

### Q4：发布后多久能看到？
A：立即生效！发布成功后会自动跳转到更新日志页面，新版本会显示在最顶部。

### Q5：可以删除已发布的版本吗？
A：目前系统不支持删除功能。如需删除，请联系技术团队。
