## 功能概述

<p style={{ fontSize: '15px', color: '#5D5D5D', lineHeight: 1.8, marginBottom: '20px' }}>
  公共频道是 Zleap 内置的优质信息源集合，借助「共享信息源」，您可以从海量信息中选择感兴趣的高价值内容，提升信息获取效率，后续可以作为创建助手、创建任务、生成报告和帖子的基础！
</p>

## 操作教程

### 进入「共享信息源」

<p style={{ fontSize: '15px', color: '#5D5D5D', lineHeight: 1.8, marginBottom: '16px' }}>
  点击「创作中心」的「信息管理」，进入「共享信息源」界面。
</p>

<div style={{ border: '1px solid #EDEDED', borderRadius: '12px', overflow: 'hidden', marginBottom: '16px' }}>
  <img src="/images/information-management/subscribe-shared-step1.png" alt="进入共享信息源" style={{ width: '100%', display: 'block' }} />
</div>

<div style={{ background: '#FFF7ED', borderLeft: '3px solid #FF8A00', padding: '16px 20px', borderRadius: '8px', marginBottom: '24px' }}>
  <p style={{ fontSize: '14px', color: '#5D5D5D', lineHeight: 1.7, margin: 0 }}>
    当前，「共享信息源」已汇聚了来自 AI、时政、商业、职场等领域的数千条优质、新鲜的信息源，持续为您提供高质量公共信息。
  </p>
</div>

### 订阅共享信息源

#### 订阅频道

<p style={{ fontSize: '15px', color: '#5D5D5D', lineHeight: 1.8, marginBottom: '12px' }}>
  选择需订阅的频道，在对应频道卡片右侧点击「加入」按钮。Zleap 支持订阅多个频道，您可任意订阅喜欢的频道。
</p>

<div style={{ border: '1px solid #EDEDED', borderRadius: '12px', overflow: 'hidden', marginBottom: '12px' }}>
  <img src="/images/information-management/subscribe-shared-step2.gif" alt="订阅频道" style={{ width: '100%', display: 'block' }} />
</div>

<p style={{ fontSize: '14px', color: '#898989', lineHeight: 1.7, marginBottom: '24px' }}>
  💡 若想取消订阅，在订阅成功后，点击卡片右侧「已加入」按钮，即可取消订阅。
</p>

#### 查看订阅内容

<p style={{ fontSize: '15px', color: '#5D5D5D', lineHeight: 1.8, marginBottom: '16px' }}>
  「加入」后，您可以在「信息源列表」的「我加入的」查看您已经订阅的所有「公共信息源」。
</p>

<div style={{ border: '1px solid #EDEDED', borderRadius: '12px', overflow: 'hidden', marginBottom: '16px' }}>
  <img src="/images/information-management/subscribe-shared-step3.png" alt="查看订阅内容" style={{ width: '100%', display: 'block' }} />
</div>

<p style={{ fontSize: '15px', color: '#5D5D5D', lineHeight: 1.8, marginBottom: '16px' }}>
  点击任一频道，即可进入其最新发布的信息列表；点击具体的「事件名称」，可跳转至原文详情页。
</p>

<div style={{ border: '1px solid #EDEDED', borderRadius: '12px', overflow: 'hidden', marginBottom: '24px' }}>
  <img src="/images/information-management/subscribe-shared-step4.gif" alt="查看信息详情" style={{ width: '100%', display: 'block' }} />
</div>

#### 使用 AI「搜索助手」检索内容

<p style={{ fontSize: '15px', color: '#5D5D5D', lineHeight: 1.8, marginBottom: '16px' }}>
  在频道信息页，您还可使用「搜索助手」进行 AI 驱动的信息检索。
</p>

<div style={{ border: '1px solid #EDEDED', borderRadius: '12px', overflow: 'hidden' }}>
