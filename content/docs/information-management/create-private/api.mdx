## 如何使用 API 接入数据？

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0 40px 0' }}>
  以「接入高德天气 API，获取广州实时天气」为例，演示如何通过 API 将第三方数据接入智跃 Zleap。
</p>

### 步骤一：获取 API

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0' }}>
  <strong>1. 找到 API 提供方</strong>
</p>

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0' }}>
  确定数据来源平台（如高德开放平台、企业内部系统等）。
</p>

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '32px 0 12px 0' }}>
  <strong>2. 申请 API 密钥</strong>
</p>

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0' }}>
  以高德天气 API 为例：
</p>

<ol style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0', paddingLeft: '20px' }}>
  <li>打开高德开放平台官网，注册 / 登录开发者账号</li>
  <li>进入「控制台」→「应用管理」→「创建应用」（命名如 "智跃天气同步"）</li>
  <li>应用创建后，点击「添加 Key」，选择「Web 服务 API」类型，提交申请</li>
</ol>

<div style={{ display: 'flex', gap: '12px', alignItems: 'center', margin: '32px 0' }}>
  <div style={{ flexShrink: 0 }}>
    <Lightbulb className="h-5 w-5" style={{ color: '#FF8800' }} />
  </div>
  <p style={{ fontSize: '16px', fontWeight: 400, color: '#1A1D23', lineHeight: '26px', margin: 0 }}>
    <strong>关键提醒：</strong>拿到 API 后，一定要问清 3 件事：是否需要密钥（token/AppID）？（安全相关，必填）；调用频率限制（比如每分钟最多调用 100 次，避免被拉黑）；数据返回格式（一般是 JSON，确保智跃 Zleap 支持）。
  </p>
</div>

## 常见问题

<div style={{ display: 'grid', gridTemplateColumns: '1fr', gap: '16px', margin: '20px 0 40px 0' }}>
  <div style={{ background: '#FFFFFF', border: '1px solid rgba(0, 0, 0, 0.04)', borderRadius: '16px', padding: '24px' }}>
    <div style={{ display: 'flex', alignItems: 'center', gap: '10px', marginBottom: '12px' }}>
      <div style={{ width: '40px', height: '40px', background: 'rgba(255, 138, 0, 0.1)', borderRadius: '10px', display: 'flex', alignItems: 'center', justifyContent: 'center', color: '#FF8A00', flexShrink: 0 }}>
        <Wrench className="h-5 w-5" />
      </div>
      <p style={{ fontSize: '17px', fontWeight: 600, color: '#1d1d1f', margin: 0 }}>调试失败怎么办？</p>
    </div>
    
    <p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0' }}>
      <strong>1. 检查 API 地址是否正确</strong>
    </p>
    <ul style={{ fontSize: '17px', color: '#86868b', lineHeight: 1.7, margin: '0 0 16px 0', paddingLeft: '20px' }}>
      <li>确认 URL 是否完整（包含 http:// 或 https://）</li>
      <li>检查是否有拼写错误</li>
      <li>确认 API 地址是否可访问（可在浏览器中直接访问测试）</li>
    </ul>
    
    <p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0' }}>
      <strong>2. 检查参数配置</strong>
    </p>
    <ul style={{ fontSize: '17px', color: '#86868b', lineHeight: 1.7, margin: '0 0 16px 0', paddingLeft: '20px' }}>
      <li>必填参数是否都已填写</li>
      <li>参数值格式是否正确（如日期格式、数字格式等）</li>
      <li>密钥（key/token）是否有效且未过期</li>
    </ul>
    
    <p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0' }}>
      <strong>3. 检查请求方式</strong>
    </p>
    <ul style={{ fontSize: '17px', color: '#86868b', lineHeight: 1.7, margin: '0 0 16px 0', paddingLeft: '20px' }}>
      <li>确认选择的请求方式（GET/POST/PUT/DELETE）与 API 文档要求一致</li>
      <li>大部分查询类 API 使用 GET 方式</li>
    </ul>
    
    <p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0' }}>
      <strong>4. 检查网络连接</strong>
    </p>
    <ul style={{ fontSize: '17px', color: '#86868b', lineHeight: 1.7, margin: 0, paddingLeft: '20px' }}>
      <li>确认网络连接正常</li>
      <li>检查是否有防火墙或代理限制</li>
      <li>确认 API 服务器是否正常运行</li>
    </ul>
  </div>
  
  <div style={{ background: '#FFFFFF', border: '1px solid rgba(0, 0, 0, 0.04)', borderRadius: '16px', padding: '24px' }}>
    <div style={{ display: 'flex', alignItems: 'center', gap: '10px', marginBottom: '12px' }}>
      <div style={{ width: '40px', height: '40px', background: 'rgba(255, 138, 0, 0.1)', borderRadius: '10px', display: 'flex', alignItems: 'center', justifyContent: 'center', color: '#FF8A00', flexShrink: 0 }}>
        <Settings className="h-5 w-5" />
      </div>
      <p style={{ fontSize: '17px', fontWeight: 600, color: '#1d1d1f', margin: 0 }}>如何设置合适的超时时间？</p>
    </div>
    
    <p style={{ fontSize: '17px', color: '#86868b', lineHeight: 1.7, margin: '12px 0 16px 0' }}>
      超时时间是指系统等待 API 响应的最长时间，超过这个时间就会认为请求失败。
    </p>
    
    <p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0' }}>
      <strong>建议设置：</strong>
    </p>
    <ul style={{ fontSize: '17px', color: '#86868b', lineHeight: 1.7, margin: '0 0 16px 0', paddingLeft: '20px' }}>
      <li><strong style={{ color: '#1d1d1f' }}>轻量级查询（如天气、汇率）</strong>：10-20 秒</li>
      <li><strong style={{ color: '#1d1d1f' }}>中等数据量（如新闻列表、商品信息）</strong>：20-30 秒</li>
      <li><strong style={{ color: '#1d1d1f' }}>大数据量或复杂查询</strong>：30-60 秒</li>
    </ul>
    
    <div style={{ display: 'flex', gap: '12px', alignItems: 'center', margin: '32px 0' }}>
      <div style={{ flexShrink: 0 }}>
        <Lightbulb className="h-5 w-5" style={{ color: '#FF8800' }} />
      </div>
      <p style={{ fontSize: '16px', fontWeight: 400, color: '#1A1D23', lineHeight: '26px', margin: 0 }}>
        <strong>提示：</strong>如果是数据量大、处理复杂的请求（如报表生成、大文件下载），可以适当调长超时时间（如 60 秒）；如果是实时性要求高的请求（如支付、订单接口），建议调短超时时间（如 10-15 秒），避免用户等待过久。
      </p>
    </div>
  </div>
</div>
