## 常见问题（FAQ）

### Q1：为什么「提交链接」按钮是灰色的？

<p style={{ fontSize: '16px', fontWeight: 400, color: '#1A1D23', lineHeight: '26px', margin: '10px 0' }}>
  <strong>A：</strong>至少有 1 条链接未通过格式 / 连通性 / 重复 / 数量校验，请根据红色提示修正后再提交。
</p>

### Q2：提示 "链接已存在" 是什么意思？

<p style={{ fontSize: '16px', fontWeight: 400, color: '#1A1D23', lineHeight: '26px', margin: '10px 0' }}>
  <strong>A：</strong>该链接在系统中已提交过（未提交或已提交均会判断），不支持重复提交。
</p>

### Q3：最多能提交几条链接？

<p style={{ fontSize: '16px', fontWeight: 400, color: '#1A1D23', lineHeight: '26px', margin: '10px 0' }}>
  <strong>A：</strong>单次最多支持 5 条。
</p>

### Q4：提示 "链接失效" 怎么办？

<p style={{ fontSize: '16px', fontWeight: 400, color: '#1A1D23', lineHeight: '26px', margin: '10px 0 32px 0' }}>
  <strong>A：</strong>请检查链接是否可在浏览器正常打开；若可打开仍提示失效，可能是网络超时或目标网站限制访问。
</p>
