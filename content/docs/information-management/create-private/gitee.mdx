## 什么是 Gitee 信息源？

通过 Gitee 信息源，您可以将 Gitee 仓库中的代码、文档、Issue、PR 等内容接入到 Zleap，让助手基于这些内容进行分析和创作。

## 使用场景

- 📝 跟踪项目开发进度和代码变更
- 🐛 监控 Issue 和 Bug 报告
- 🔄 分析 Pull Request 和代码审查
- 📚 整理项目文档和 README
