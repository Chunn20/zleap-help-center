## 完整操作步骤

### 步骤1：进入创建页面

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, marginBottom: '16px' }}>
  在 Zleap 中进入【信息管理】 - 【创建信息源】 - 【Gitee】，选择 Gitee 接入方式。
</p>

<div style={{ margin: '24px auto', borderRadius: '12px', overflow: 'hidden', border: '1px solid rgba(0, 0, 0, 0.04)', maxWidth: '100%', width: '100%', height: '400px', padding: '24px', display: 'flex', alignItems: 'center', justifyContent: 'center', background: '#FAFAFA', cursor: 'pointer', transition: 'transform 0.2s' }}>
  <img src="/images/information-management/create-private/gitee/select-gitee.gif" alt="选择Gitee接入" style={{ maxWidth: '100%', maxHeight: '100%', width: 'auto', height: 'auto', display: 'block', objectFit: 'contain' }} />
</div>

### 步骤2：一键授权连接

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, marginBottom: '16px' }}>
  页面会自动跳转到 Gitee 官方授权页。若未登录，先输入 Gitee 账号密码完成登录。确认授权给 Zleap（仅授予读取仓库、用户信息的只读权限），点击【授权】。授权成功后自动跳回 Zleap，页面状态从【待授权】变为【已授权】。
</p>



### 步骤3：补充信息源配置

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, marginBottom: '16px' }}>
  【信息名称】给信息源起一个辨识度高的名字，比如「公司 Gitee 代码库」。
</p>
<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, marginBottom: '16px' }}>
  【信息要点】填写用途说明，比如「同步公司前端代码仓库，用于 AI 代码问答」。
</p>
<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, marginBottom: '16px' }}>
  【启用状态】默认开启（开启后才能被 AI 应用关联使用）。
</p>
<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, marginBottom: '16px' }}>
  【首次同步】可自定义首次同步时间，默认是当前时间。
</p>
<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, marginBottom: '16px' }}>
  【同步频率】设置数据自动同步的间隔（例：3 小时，即每 3 小时拉取一次 Gitee 最新数据）。
</p>



### 步骤4：选择同步仓库 / 分支

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, marginBottom: '16px' }}>
  授权成功后，页面会加载你 Gitee 下的所有仓库，勾选需要同步给 Zleap 的仓库、指定分支，避免同步无关数据。
</p>


### 步骤5：提交完成接入

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, marginBottom: '16px' }}>
  确认所有配置无误后，点击右上角【提交信息】，完成 Gitee 信息源的接入。
</p>

<div style={{ background: '#FFF9F0', border: '1px solid #FFE9CC', borderRadius: '12px', padding: '20px 24px', margin: '32px 0' }}>

    后续可在【信息源列表】查看同步状态，在【信息源日志】排查同步问题。
  </p>
</div>

## 关键说明 & 常见问题

### 权限说明

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, marginBottom: '16px' }}>
  授权后 Zleap 仅拥有只读权限，只能读取代码 / 文档数据，无法修改、删除你的 Gitee 内容。可随时在 Gitee【设置 → 第三方应用】中找到 Zleap 应用，点击【撤销】终止授权。
</p>
