## 本地文档上传

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0' }}>
  您可以将本地文档上传至 Zleap，构建您的专属信息源。完成上传后，此信息源即可作为 AI 助手创建任务和报告的信息基础，您还可以使用AI工具精准搜索该信息源中的内容。
</p>

### 第一步：进入创建页面
<ul style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.8, paddingLeft: '20px' }}>
  <li>登录 Zleap，点击「创建信息源」，点击「文档上传」。</li>
</ul>

<div style={{ margin: '24px auto', borderRadius: '12px', overflow: 'hidden', border: '1px solid rgba(0, 0, 0, 0.04)', maxWidth: '100%', width: '100%', height: '400px', padding: '24px', display: 'flex', alignItems: 'center', justifyContent: 'center', background: '#FAFAFA', cursor: 'pointer', transition: 'transform 0.2s' }}>
  <img src="/images/information-management/create-private/document-upload/step1.gif" alt="进入创建页面" style={{ maxWidth: '100%', maxHeight: '100%', width: 'auto', height: 'auto', display: 'block', objectFit: 'contain' }} />
</div>

### 第二步：上传并设置文档
<ul style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.8, paddingLeft: '20px' }}>
  <li>在 "上传信息源" 页面，点击「选择文件」，上传您的文档。目前，Zleap 支持 pdf、txt、md、docx 等格式。</li>
  <li>文件上传成功后，在「信息名称」字段中，输入信息源名称。</li>
  <li>打开「启用状态」则后续创建助手和任务的时候可以使用该信息源。</li>
  <li>确认信息无误后，点击右上角的「提交信息」按钮。</li>
</ul>

<div style={{ margin: '24px auto', borderRadius: '12px', overflow: 'hidden', border: '1px solid rgba(0, 0, 0, 0.04)', maxWidth: '100%', width: '100%', height: '400px', padding: '24px', display: 'flex', alignItems: 'center', justifyContent: 'center', background: '#FAFAFA', cursor: 'pointer', transition: 'transform 0.2s' }}>
  <img src="/images/information-management/create-private/document-upload/step2.gif" alt="上传并设置文档" style={{ maxWidth: '100%', maxHeight: '100%', width: 'auto', height: 'auto', display: 'block', objectFit: 'contain' }} />
</div>

### 第三步：完成创建
<ul style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.8, paddingLeft: '20px' }}>
  <li>提交成功后，页面会提示 "信息源创建完成"。</li>
  <li>方式一：可以选择「创建助手」，让助手以该信息源作为背景信息。</li>
  <li>方式二：可以选择「创建任务」，基于信息源定期发布动态报告。</li>
</ul>

## 粘贴链接

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0' }}>

</p>

### 第一步：进入创建页面
<ul style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.8, paddingLeft: '20px' }}>

<div style={{ margin: '24px auto', borderRadius: '12px', overflow: 'hidden', border: '1px solid rgba(0, 0, 0, 0.04)', maxWidth: '100%', width: '100%', height: '400px', padding: '24px', display: 'flex', alignItems: 'center', justifyContent: 'center', background: '#FAFAFA', cursor: 'pointer', transition: 'transform 0.2s' }}>

### 第二步：粘贴链接
<ul style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.8, paddingLeft: '20px' }}>
  <li>在 "上传信息源" 页面，直接在输入框中粘贴以 http:// 或 https:// 开头的 URL 地址。</li>
