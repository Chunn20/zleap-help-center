<div className="gitlab-section">

    <li>支持 CI/CD、Issue 跟踪、代码审查等完整的 DevOps 工作流</li>
    <li>可以自托管部署，也可以使用 GitLab.com 云服务</li>
  </ul>

  <p className="gitlab-intro">

  </p>

<h3 id="permissions" style={{ fontSize: '18px', fontWeight: 600, color: 'var(--text-primary, #0D131A)', marginBottom: '16px', marginTop: '24px' }}>
  关于权限范围（Scopes）
</h3>

<div className="gitlab-section">
  <p className="gitlab-intro">
    在 GitLab 中创建 OAuth 应用时，需要选择合适的权限范围：
  </p>

  <ul style={{ fontSize: '14px', color: 'var(--text-secondary, #5D5D5D)', lineHeight: 1.8, paddingLeft: '20px', marginBottom: '16px' }}>
    <li><strong>read_user：</strong>读取用户基本信息（必需）</li>
    <li><strong>read_repository：</strong>读取仓库代码和文件（推荐）</li>
    <li><strong>read_api：</strong>读取 API 数据，包括 Issue、MR 等（推荐）</li>
  </ul>

  <div className="gitlab-warning">
    <p style={{ fontSize: '14px', color: 'var(--text-secondary, #5D5D5D)', lineHeight: 1.7, margin: 0 }}>
      <AlertCircle className="inline h-4 w-4 mr-1 text-[#FF8A00]" />
      <strong>安全提示：</strong>只授予 Zleap 必要的权限，避免授予写入权限（如 <code>write_repository</code>、<code>api</code>）
    </p>
  </div>
</div>

<h3 id="troubleshooting" style={{ fontSize: '18px', fontWeight: 600, color: 'var(--text-primary, #0D131A)', marginBottom: '16px', marginTop: '24px' }}>
  常见错误排查
</h3>

<div className="gitlab-section">
  <div className="gitlab-highlight">
    <h4 style={{ fontSize: '16px', fontWeight: 600, color: 'var(--text-primary, #0D131A)', marginBottom: '12px', marginTop: 0 }}>
      <AlertCircle className="inline h-5 w-5 mr-2 text-[#FF8A00]" />
      错误 1：Redirect URI mismatch
    </h4>
    <p style={{ fontSize: '14px', color: 'var(--text-secondary, #5D5D5D)', lineHeight: 1.8, marginBottom: '8px' }}>
      <strong>原因：</strong>GitLab OAuth 应用中配置的回调地址与 Zleap 提供的不一致
    </p>
    <p style={{ fontSize: '14px', color: 'var(--text-secondary, #5D5D5D)', lineHeight: 1.8, marginBottom: '0' }}>
      <strong>解决：</strong>检查并确保 GitLab 应用配置中的 Redirect URI 与 Zleap 显示的完全一致（包括协议、域名、路径）
    </p>
  </div>

  <div className="gitlab-highlight">
    <h4 style={{ fontSize: '16px', fontWeight: 600, color: 'var(--text-primary, #0D131A)', marginBottom: '12px', marginTop: 0 }}>
      <AlertCircle className="inline h-5 w-5 mr-2 text-[#FF8A00]" />
      错误 2：Invalid client credentials
    </h4>
    <p style={{ fontSize: '14px', color: 'var(--text-secondary, #5D5D5D)', lineHeight: 1.8, marginBottom: '8px' }}>
      <strong>原因：</strong>Application ID 或 Secret 填写错误
    </p>
    <p style={{ fontSize: '14px', color: 'var(--text-secondary, #5D5D5D)', lineHeight: 1.8, marginBottom: '0' }}>
      <strong>解决：</strong>重新从 GitLab OAuth 应用页面复制 Application ID 和 Secret，确保没有多余的空格或字符
    </p>
  </div>

  <div className="gitlab-highlight">
    <h4 style={{ fontSize: '16px', fontWeight: 600, color: 'var(--text-primary, #0D131A)', marginBottom: '12px', marginTop: 0 }}>
      <AlertCircle className="inline h-5 w-5 mr-2 text-[#FF8A00]" />
      错误 3：Connection timeout（自托管实例）
    </h4>
    <p style={{ fontSize: '14px', color: 'var(--text-secondary, #5D5D5D)', lineHeight: 1.8, marginBottom: '8px' }}>
      <strong>原因：</strong>GitLab 实例地址无法访问或网络不通
    </p>
    <p style={{ fontSize: '14px', color: 'var(--text-secondary, #5D5D5D)', lineHeight: 1.8, marginBottom: '0' }}>
      <strong>解决：</strong>确认 GitLab 实例地址正确，且 Zleap 服务器可以访问该地址（检查防火墙、VPN 等网络配置）
    </p>
  </div>

  <div className="gitlab-highlight">
    <h4 style={{ fontSize: '16px', fontWeight: 600, color: 'var(--text-primary, #0D131A)', marginBottom: '12px', marginTop: 0 }}>
      <AlertCircle className="inline h-5 w-5 mr-2 text-[#FF8A00]" />
      错误 4：Insufficient permissions
    </h4>
    <p style={{ fontSize: '14px', color: 'var(--text-secondary, #5D5D5D)', lineHeight: 1.8, marginBottom: '8px' }}>
      <strong>原因：</strong>OAuth 应用的权限范围不足
    </p>
    <p style={{ fontSize: '14px', color: 'var(--text-secondary, #5D5D5D)', lineHeight: 1.8, marginBottom: '0' }}>
      <strong>解决：</strong>返回 GitLab OAuth 应用配置，勾选必要的 Scopes（至少需要 <code>read_user</code> 和 <code>read_repository</code>）
    </p>
  </div>
</div>
