## 如何使用 RSS 作为信息源？

### 步骤1：添加 RSS 链接

<p style={{ fontSize: '17px', color: '#86868b', lineHeight: 1.7, marginBottom: '16px' }}>
  输入 http:// 或 https:// 开头的 RSS URL 地址。可换行输入多个链接，批量添加订阅源。
</p>

### 步骤2：验证链接

<p style={{ fontSize: '17px', color: '#86868b', lineHeight: 1.7, marginBottom: '16px' }}>
  点击「验证链接」按钮，系统会自动检测链接是否有效。请确保输入的是有效的 RSS 源地址。如果验证失败，请检查：
</p>

<ul style={{ fontSize: '17px', color: '#86868b', lineHeight: 1.8, paddingLeft: '20px', marginBottom: '16px' }}>
  <li>URL 格式是否正确</li>
  <li>RSS 源是否可访问</li>
  <li>网络连接是否正常</li>
</ul>

### 步骤3：填写信息并提交

<p style={{ fontSize: '17px', color: '#86868b', lineHeight: 1.7, marginBottom: '16px' }}>
  填写「信息名称」等必要信息和选填信息。
</p>
