## 核心价值

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '0 0 16px 0' }}>
  <strong style={{ color: '#1d1d1f' }}>信息搜索</strong>是 Zleap 基于 SAG（语义增强检索，Semantic-Augmented Generation）技术打造的私有知识库智能搜索入口，是你调用 AI 知识库的核心窗口。
</p>

<ul style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '0 0 60px 0', paddingLeft: '20px' }}>
  <li><strong>统一检索入口</strong>：在自己搭建的所有信息源（文档、网页、代码库、聊天记录等）中，像用搜索引擎一样精准、高效地找到需要的信息</li>
  <li><strong>语义理解能力</strong>：区别于传统关键词匹配，SAG 技术能深度理解你提问的真实意图，而不是只匹配字面文字</li>
  <li><strong>智能结果呈现</strong>：提供「AI 总结 + 原文溯源 + 知识图谱」三层呈现，快速获取关键信息的同时保证可追溯性</li>
  <li><strong>公私数据分离</strong>：支持公共知识库和个人知识库分别检索，既能协作共享，又能保护个人隐私</li>
</ul>

## 核心功能

### 功能一：语义化精准搜索（SAG 技术核心价值）

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0' }}>
  区别于传统关键词匹配，SAG 技术能深度理解你提问的真实意图，而不是只匹配字面文字：
</p>

<ul style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0 16px 0', paddingLeft: '20px' }}>
  <li>比如你输入「美国如何主导核军控规则？」，系统会自动理解问题的核心逻辑，从你的知识库中精准召回相关的政策、分析、报告</li>
  <li>哪怕原文没有完全一致的关键词，也能匹配到语义相关的内容</li>
  <li>解决了传统搜索"搜不全、搜不准、漏结果"的痛点</li>
</ul>

### 功能二：双场景检索，覆盖公私数据

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0' }}>
  页面分「公共」「我的」两个标签，满足不同使用需求：
</p>

<ul style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0 16px 0', paddingLeft: '20px' }}>
  <li><strong>公共</strong>：检索团队共享的公共知识库内容，适合查公司公开的产品文档、行业资料、共享信息源</li>
  <li><strong>我的</strong>：检索你个人创建 / 加入的私有信息源，比如个人工作笔记、专属项目资料、私密文档，保障数据安全</li>
</ul>
