## 功能概述

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0 40px 0' }}>
  Zleap 提供完整的界面语言切换功能，旨在为全球用户与团队提供更友好、个性化的产品使用体验。目前支持<strong>简体中文</strong>与 <strong>English</strong>，您可以根据个人偏好随时更改。
</p>

## 操作步骤

### 网页版切换语言

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0' }}>
  <strong>步骤 1：找到语言切换入口</strong>
</p>

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0 16px 0' }}>
  登录您的 Zleap 账号，进入创作中心任意模块，可在界面右上角找到「语言切换」图标。
</p>

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '32px 0 12px 0' }}>
  <strong>步骤 2：选择目标语言</strong>
</p>

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0 16px 0' }}>
  点击该图标，系统会弹出语言选择菜单，选择您希望使用的语言「中文」或「English」。
</p>

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '32px 0 12px 0' }}>
  <strong>步骤 3：界面自动更新</strong>
</p>

<p style={{ fontSize: '17px', color: '#1d1d1f', lineHeight: 1.7, margin: '12px 0 16px 0' }}>
  点击后，页面将自动刷新，界面文字将更新为您选择的语言。
</p>
