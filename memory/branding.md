name: Fumadocs Visual Branding
description: Implements project-specific visual branding using primary #FEF9E6 and accent #AFDCFF colors.
type: feedback
---

- Use #FEF9E6 as the primary background color for the application.
- Use #AFDCFF as the accent/transition color for active/hover states.
- Ensure text contrast is maintained against the #FEF9E6 background using dark grey instead of pure black.
- Apply high levels of whitespace for a professional look.
- Use rounded-xl (16px) for card elements.
- Use shadow-sm/shadow-md for subtle layering.

**Why:** The user wants the documentation to feel like a mature, professional product rather than a "hand-crafted" project.
**How to apply:** Apply these colors and spacing principles across layout components, CSS variables, and page templates.
